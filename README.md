## Description

This p5.js project demonstrates a reflective camera-style experience where a player can navigate through a heavenly
scene/world that is larger than the viewport. This project uses "lerp" to smooth camera movements and emphasize the slow
relaxing theme of the expereince

---

## Learning Goals

- Understand how to seperate world state from view state
- Load images into p5.js projects and use them in a more complex manner
- Control camera and player movement by adding bounds
